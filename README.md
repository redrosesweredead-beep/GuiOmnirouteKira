Код для оптимизированной работой с claude sonnet через omniroute proxy. Оптимизированный GUI  с возможностью быстрой локальной установки кода
Сам omniroute настраивал через видео https://www.youtube.com/watch?v=5-qW_lSTRuI
В коде файла нужно только вставить ваш ключ API все остальное уже готово к использованию
